# GeoIP Database Required

To enable GeoIP features, please download the MaxMind GeoLite2 City and ASN databases and place them here.

Files required:
- `ipcity.mmdb` (Rename GeoLite2-City.mmdb to this)
- `iporg.mmdb` (Rename GeoLite2-ASN.mmdb to this)

Get them from: https://dev.maxmind.com/geoip/geolite2-free-geolocation-data
